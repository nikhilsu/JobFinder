# Job Finder

Finds Jobs given Job description.

## Installation

```
 python >= 2.7
 selenium webdriver (Chrome)
 phantomJS webdriver
 python-selenium bindings
 texttable
```

## Usage
Headless mode :-
```
python main.py --headless
```
Non-headless mode :- 
```
python main.py
```
